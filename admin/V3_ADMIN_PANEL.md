## üîß Installation Requirements

### Server Prerequisites
- PHP 7.4 or higher
- MySQL 5.7 or higher
- Write permissions for backup directory
- GD library for image processing (optional)

### Setup Process
1. Upload admin folder to server
2. Ensure proper file permissions (755 for directories, 644 for files)
3. Access admin panel through `/admin/dashboard.php`
4. Login with admin credentials
5. Configure initial settings
